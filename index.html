Create a new repository on the command line

  1. Entra nella cartella del progetto
  2. git init
  3. git add .
  4. git commit -m "first commit"
  5. git branch -M main
  6. git remote add origin https://github.com/gladiator95/lezione2.git
  7. git push -u origin main
